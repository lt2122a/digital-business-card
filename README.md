## 一、作品名称
Personal Digital Business Card（个人数字名片）

## 二、作品简介
本作品是一个基于 HTML + CSS 开发的响应式个人数字名片，用于展示个人基本信息、专业背景、自我介绍、兴趣与技能标签，并集成外部社交与博客链接。界面采用现代化渐变设计、卡片式布局与动态交互效果，支持电脑、手机等多端自适应显示，是一个结构完整、美观规范的前端软件作品。

## 三、页面主要内容说明
1. 个人信息区：头像、姓名、学号、专业、联系方式统一展示
2. 自我介绍区：个人学习经历、技术方向、职业规划
3. 标签区：兴趣标签、技能标签，支持悬浮动态效果
4. 拓展链接区：Gitee 仓库、技术博客外部跳转
5. 整体采用卡片式设计，具备渐变、阴影、悬浮动效等视觉增强效果

## 七、AI 辅助开发说明
### 1. 是否使用 AI
是，本项目在开发过程中使用了 AI 辅助开发。

### 2. 使用了哪些工具
- 豆包 AI
- 通义千问

### 3. 关键提示词（2-4 条）
1. 帮我写一个高端大气的个人数字名片 HTML + CSS
2. 个人主页卡片式设计，渐变背景，悬浮动效
3. 帮我生成符合作业要求的 README.md
4. 帮我优化 HTML 结构与 CSS 样式

### 4. AI 主要帮助了哪些部分
- 提供完整的 HTML 页面结构与模块化布局思路
- 生成现代化 CSS 样式、渐变、动画、响应式代码
- 协助完成 README.md 规范写作
- 优化界面排版、配色与交互效果

### 5. 对 AI 结果做了哪些修改
- 修改为个人真实信息：姓名、学号、专业、联系方式
- 替换为个人头像与真实社交链接
- 调整配色、间距、字体等样式使其更符合个人风格
- 优化代码结构，使其更规范、易读

### 6. 认为 AI 最不可靠的一点
AI 生成的代码有时会出现样式冲突、类名重复、布局兼容性问题，不能直接完全使用，必须人工检查、调整与优化，无法完全替代人工开发与调试。
